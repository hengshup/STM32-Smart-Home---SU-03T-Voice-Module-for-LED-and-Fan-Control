这是一个用STM32F103C8T6芯片控制的智能家居，用su-03t语音模块控制LED和直流电机，oled显示状态
具体指令可查看代码注释。
要注意的是直流电机需要用TB6612FNG点击驱动模块来驱动，接线如下：


STM32                 OLED
PB8      -----------  SCL
PB9      -----------  SDA
                      电机驱动模块
PA2      -----------  PWMA
PA4      -----------  AIN1
PA5      -----------  AIN2
                      SU-03T
PA10     -----------  RXD
PA9     -----------   TXD


直流电机 --------- 电机驱动模块A01
直流电机 --------- 电机驱动模块A02

VM       --------- 5V
VCC      --------- 3.3v
